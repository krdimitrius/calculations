Очень важно чтобы файлы *.dat находились в том же каталоге с Qbasic.exe, иначе программа будет "ругаться".
Если у Вас DOS не русифицирована, то при помощи файла rksv.com c ключами 2 S I
Т.е. " rksv.com 2 S I ".
Затем запустите программу qbasic.exe
Откройте файл prog.bas, OK
И нажмите F5 или shift+F5 для запуска программы
